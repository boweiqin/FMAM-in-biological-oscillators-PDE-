The codes in this repository are used to generate the data in Figure 1e-f, which corresponds to the case:
suppression the protein concentration in the cancer network.

Please run the script in the following sequence:

1.   CNAM1.m     % codes for generating solution in time interval 0<t<50
2.   CNAM2.m     % codes for generating solution in time interval 50<t<100
3.   CNAM3.m     % codes for generating solution in time interval 100<t<150
4.   CNAM4.m     % codes for generating solution in time interval 150<t<200
5.   CNAM5.m     % codes for generating solution in time interval 200<t<250
6.   CNAM6.m     % codes for generating solution in time interval 250<t<300
7.   MergeSol.m  % codes for merging all solutions together
8.   PlotFig.m      % codes for generating the figures
