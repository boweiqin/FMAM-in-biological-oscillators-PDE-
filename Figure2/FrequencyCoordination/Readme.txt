The codes in this repository are used to generate the data in Figure 1a-d, which corresponds to the case:
independent frequency coordination in the cancer network.

Please run the script in the following sequence:

1.   CNFM1.m     % codes for generating solution in time interval 0<t<50
2.   CNFM2.m     % codes for generating solution in time interval 50<t<100
3.   CNFM3.m     % codes for generating solution in time interval 100<t<150
4.   CNFM4.m     % codes for generating solution in time interval 150<t<200
5.   CNFM5.m     % codes for generating solution in time interval 200<t<250
6.   CNFM6.m     % codes for generating solution in time interval 250<t<300
7.   MergeSol.m  % codes for merging all solutions together
8.   PlotFig.m      % codes for generating the figures
